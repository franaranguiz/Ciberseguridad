Amenazas Cibernéticas Comunes

Phishing

Engaños por correo o mensajes para robar datos personales.

El phishing consiste en suplantar la identidad de una entidad confiable para obtener datos como contraseñas o cuentas bancarias.
Ransomware

Secuestro de datos con exigencia de pago para liberarlos.

Este tipo de software malicioso bloquea el acceso a los archivos hasta que se pague un rescate, generalmente en criptomonedas.
Malware

Software diseñado para dañar o infiltrarse en un sistema.

Puede tomar la forma de virus, troyanos, spyware y más. Infecta sistemas para robar información o dañarlos.
Ataques DDoS

Colapsar un servidor mediante múltiples solicitudes simultáneas.

Se saturan los recursos de un servidor desde múltiples equipos, haciendo que el sitio sea inaccesible.

Casos Reales

En 2023, miles de personas recibieron correos falsos del “Banco Nacional” solicitando actualizar sus datos. El enlace redirigía a una web falsa.

Varios hospitales europeos fueron atacados con ransomware, paralizando sistemas de atención médica hasta que se pagaron rescates.

Servicios como PlayStation Network y Xbox Live fueron atacados por grupos hackers que los dejaron fuera de línea durante horas.
